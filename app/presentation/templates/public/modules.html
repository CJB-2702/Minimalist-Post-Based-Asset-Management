EBAMS Modules

Integrated modules working together for complete asset management

Core Module

Foundation for all asset management operations

The Core Module provides the foundational data models and business logic for users, locations, asset types, make/models, assets, and events. It handles user management with role-based access control, location hierarchy management, asset type and make/model categorization, asset lifecycle tracking, and comprehensive event logging with audit trails.

Key Features:
  • User management and authentication
  • Role-based access control
  • Location hierarchy management
  • Asset type categorization
  • Make and model tracking
  • Asset lifecycle management
  • Event logging and audit trails
  • Comments and attachments

Assets Module

Flexible detail management for asset-specific information

Extends core assets with a flexible detail table system for asset-specific and model-specific information. Create dynamic detail tables, automatically populate detail rows on asset creation, and configure detail table assignments based on asset types and models. Supports both asset-level and model-level details with factory-based creation patterns.

Key Features:
  • Dynamic detail table creation
  • Automatic detail row generation
  • Configuration-driven assignments
  • Asset-level detail tracking
  • Model-level detail templates
  • Factory-based creation
  • Custom field definitions
  • SQLAlchemy event listeners

Maintenance Module

Comprehensive maintenance management with template-based workflows

Provides complete maintenance event management with template-based workflows and three role-based portals. The module follows a three-tier hierarchy: Base Items (actual work), Template Items (reusable blueprints), and Proto Templates (generic action library). Features include maintenance event tracking, template versioning, action item management, part demand tracking, and specialized portals for technicians, managers, and fleet administrators.

Key Features:
  • Template-based maintenance procedures
  • Action item checklists
  • Part demand tracking and approval
  • Tool requirement management
  • Maintenance scheduling
  • Technician workflow portal
  • Manager planning portal
  • Fleet administrator dashboard
  • Template versioning
  • Maintenance history tracking

Inventory Module

Manage parts, stock levels, and procurement with demand planning

Comprehensive inventory management for parts, tools, stock levels, and procurement processes with demand-driven purchasing. Track inventory across multiple locations, manage purchase orders, process arrivals and receiving, link parts to maintenance events, and optimize stock levels. Includes supplier management, movement tracking, and relocation request workflows.

Key Features:
  • Multi-location inventory tracking
  • Part and tool management
  • Stock level optimization
  • Purchase order management
  • Arrival processing
  • Movement tracking
  • Supplier management
  • Part demand viewing
  • PO-to-demand linking
  • Relocation request workflow

Dispatching Module

Vehicle dispatch management through dual-portal workflow

Manage vehicle dispatch requests and assignments through a dual-portal workflow system. The Request Portal allows end users to create vehicle requests with detailed specifications and track approval status. The Dispatch Portal enables dispatchers to view all requests, visualize available assets matching criteria, and assign dispatch outcomes including approvals, denials, contracts, and reimbursements.

Key Features:
  • Request creation with specifications
  • Approval status tracking
  • Outcome form management
  • Dispatch request viewing
  • Asset availability visualization
  • Multiple outcome types
  • Status tracking and updates
  • Multi-level approval workflows
